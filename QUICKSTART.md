# Швидкий Старт

## Installation & Setup

### Крок 1: Встановлення залежностей

```bash
npm install
```

### Крок 2: Запуск сервера розробки

```bash
npm run dev
```

Додаток буде доступний за адресою **http://localhost:3000**

### Крок 3: Відкрийте у браузері

Перейдіть за посиланням та почніть користуватися сервісом!

## 💡 Tips & Tricks

✨ **Темний Режим** - Натисніть іконку місяця в заголовку щоб увімкнути темний режим  
📍 **Мої Укриття** - Натисніть кнопку з позначкою щоб знайти укриття поблизу вас  
❤️ **Обрані** - Натисніть іконку сердечка щоб додати укриття до обраних  
🔍 **Пошук** - Почніть вводити назву укриття для швидкого пошуку  

## 🔧 Production Build

Щоб створити оптимізований білд для production:

```bash
npm run build
```

Білд буде в папці `dist/` і готовий до розгортання на сервері.
