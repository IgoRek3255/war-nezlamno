# WarNezlamno - Карта Укриттів України

Сучасна веб-платформа для пошуку та перегляду укриттів та місць безпеки в Україні.

![React](https://img.shields.io/badge/React-18.2-blue)
![Tailwind CSS](https://img.shields.io/badge/Tailwind%20CSS-3.4-06B6D4)
![Vite](https://img.shields.io/badge/Vite-5.0-646CFF)

## 🌟 Основні Можливості

- ✅ **Інтерактивна Карта** - Переглядайте укриття на интерактивній карті з Leaflet
- ❤️ **Обрані Укриття** - Зберігайте улюблені укриття для швидкого доступу
- 🔍 **Розширений Пошук** - Пошук за назвою, адресою та фільтрація за типом
- 📍 **Геолокація** - Знайдіть найближчі укриття до вас
- ⭐ **Рейтинги** - Перегляньте рейтинги та відгуки користувачів
- 🌙 **Темний Режим** - Зручний темний режим для комфорту
- 📱 **Адаптивний Дизайн** - Сумісність з усіма пристроями
- ⚡ **Швидконавантженість** - Оптимізована продуктивність

## 🛠️ Технологічний Стек

### Frontend
- **React 18** - Сучасна UI бібліотека
- **React Router v6** - Маршрутизація додатку
- **Tailwind CSS** - Утилітарний CSS фреймворк
- **Leaflet** - Для інтерактивної карти
- **React Leaflet** - React компоненти для Leaflet
- **Zustand** - Управління станом
- **Lucide React** - SVG іконки

### Build Tools
- **Vite** - Швидкий build tool та сервер розробки
- **PostCSS** - Обробка CSS
- **Autoprefixer** - Додавання префіксів CSS

## 📦 Встановлення

### Вимоги
- Node.js >= 16.0.0
- npm або yarn

### Кроки встановлення

1. **Клонуйте репозиторій**
```bash
git clone https://github.com/IgoRek3255/war-nezlamno.git
cd war-nezlamno
```

2. **Встановіть залежності**
```bash
npm install
```

3. **Запустіть сервер розробки**
```bash
npm run dev
```

Додаток буде доступний за адресою `http://localhost:3000`

## 🚀 Production Build

```bash
npm run build
```

Оптимізований білд буде в папці `dist/`

Preview production build:
```bash
npm run preview
```

## 🌐 Деплой на Github Pages

1. **Створіть репозиторій на Github** з назвою `war-nezlamno` (або змініть `base` в `vite.config.js` відповідно)

2. **Додайте віддалений репозиторій**
```bash
git remote add origin https://github.com/IgoRek3255/war-nezlamno.git
```

3. **Запуште код**
```bash
git add .
git commit -m "Initial commit"
git push -u origin main
```

4. **Задеплойте на Github Pages**
```bash
npm run deploy
```

5. **Налаштуйте Github Pages**
   - Перейдіть до Settings > Pages у вашому репозиторії
   - Виберіть Source: "Deploy from a branch"
   - Branch: `gh-pages` / `root`
   - Збережіть

Ваш сайт буде доступний за адресою: `https://yourusername.github.io/war-nezlamno/`

```
src/
├── pages/
│   ├── Home.jsx          # Головна сторінка
│   ├── Map.jsx           # Сторінка з картою укриттів
│   ├── Favorites.jsx     # Сторінка обраних укриттів
│   └── AboutUs.jsx       # Сторінка про нас
├── components/
│   ├── Header.jsx        # Навігаційний заголовок
│   ├── Footer.jsx        # Нижній колонтитул
│   └── ShelterCard.jsx   # Карточка укриття
├── App.jsx               # Головний компонент
├── store.js              # Zustand store для стану
├── data.js               # Дані укриттів та утилітарні функції
├── globals.css           # Глобальні стилі
└── main.jsx              # Entry point
```

## 💾 Локальне Зберігання

Додаток використовує `localStorage` для збереження:
- Обраних укриттів
- Режиму темної теми
- Параметрів користувача

## 📊 Унікальні Фічі

1. **Proximity Search** - Автоматичне сортування укриттів за відстанню від користувача
2. **Real-time Ratings** - Рейтинги та відгуки в реальному часі
3. **Dark Mode Toggle** - Зручне перемиканні теми з один клік
4. **Favorite Management** - Система обраних з персистентним зберіганням
5. **Advanced Filtering** - Фільтрація за типом, місткістю та локацією
6. **Responsive Maps** - Адаптивна карта для мобільних пристроїв

## 🔒 Безпека та Приватність

- Всі дані проходять локальну обробку на стороні клієнта
- Геоданні користувача не відправляються на сервер
- HTTPS рекомендується для production
- Конфіденційність даних користувача

## 🐛 Відома Проблеми

- Геолокація не доступна в деяких браузерах приватного режиму
- Leaflet может потребовать коефіцієнту для дистрибутиву
